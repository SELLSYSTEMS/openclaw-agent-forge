# Workspace

This is the default agent workspace configured for the local OpenClaw runtime.

Use this directory for project work that should stay isolated from other agents.

The Markdown files in this directory are starter prompts and continuity templates for a personal OpenClaw workspace:

- `AGENTS.md`
- `BOOTSTRAP.md`
- `HEARTBEAT.md`
- `IDENTITY.md`
- `SOUL.md`
- `TOOLS.md`
- `USER.md`

Keep them local-first and customize them deliberately.
